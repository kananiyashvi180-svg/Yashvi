Comparison Table of HTML and CSS Features
Feature HTML CSS
Purpose Defines the structure and content of a webpage. Controls the design, layout, and visual style of a webpage.
Usage Uses tags like <h1>, <p>, <div> to organize content. Uses selectors and properties to style elements (e.g., color, font-size).
File Extension .html or .htm .css
Interaction Can include text, images, videos, and forms. Cannot add content, only modifies how it appears.
